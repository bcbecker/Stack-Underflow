About helpme_world

helpme_world is a codecentric help forum for developers and hobbyist programmers of all skill levels to come together, collaborate, learn, and assist each other in their programming journeys.

To get started collaborating with others, navigate to the register page. If you already have an account, login to get codin'!



We're working hard to add some new features to helpme_world!

Events calendar: Users can plan group discussions and view other upcoming events put on by fellow helpme_world users. Users will be able to choose to have reminder emails sent to them about events they are interested in attending.

pairUp: One of the most important skills and helpful ways for aspiring developers to learn is working in teams. pairUp will serve as a post board, where users interested in doing group work or pair programming can find like-minded users and collaborate.
