About Page

stackUnderflow is a codecentric help forum for developers and hobbyist programmers of all skill levels to come together, collaborate, learn, and assist each other in their programming journeys.

To get started collaborating with others, navigate to the register page. If you already have an account, login to get crackin'!



We're working hard to add some new features to stackUnderflow!

Announcements: In the announcements section, stackUnderflow admins will post important information about upcoming events, changes to the site, and polls to gauge interest in new features.

Events calendar: Users can plan group discussions and view other upcoming events put on by fellow stackUnderflow users. Users will be able to choose to have reminder emails sent to them about events they are interested in attending.

stackUp: stackUp will be a section of the site specifically dedicated to posting coding challenges, such as data structure and algorithm-based coding tests.

pairUp: One of the most important skills and helpful ways for aspiring developers to learn is working in teams. pairUp will serve as a post board, where users interested in doing group work or pair programming can find like-minded users and collaborate.
